Introduction

C is a powerful general-purpose programming language. It can be used to develop software like operating systems, databases, compilers, and so on. C programming is an excellent language to learn to program for beginners.

A successor to the programming language B, C was originally developed at Bell Labs by Dennis Ritchie between 1972 and 1973 to construct utilities running on Unix.

Data Types

Data types in c refer to an extensive system used for declaring variables or functions of different types. The type of a variable determines how much space it occupies in storage and how the bit pattern stored is interpreted.

ANSI C provides three types of data types:

  1. Primary(Built-in) Data Types
    1. int
    2. char
    3. float
    4. double
    5. void
  2. Derived Data Types
    1. pointer
    2. arrays
  3. User Defined Data Types
    1. struct
    2. union

 #include <stdio.h>

 }
Variables

In programming, a variable is a container (storage area) to hold data. To indicate the storage area, each variable should be given a unique name (identifier). Variable names are just the symbolic representation of a memory location. For example:

int playerScore = 95;

Here, playerScore is a variable of int type. Here, the variable is assigned an integer value 95. The value of a variable can be changed, hence the name variable.

C is a strongly typed language. This means that the variable type cannot be changed once it is declared

The general method of declaring variable is:

type variable_name = value;

Rules for naming a variable:

  1. A variable name can only have letters (both uppercase and lowercase letters), digits and underscore.
  2. The first letter of a variable should be either a letter or an underscore.
  3. There is no rule on how long a variable name (identifier) can be. However, you may run into problems in some compilers if the variable name is longer than 31 characters.

An Example is given below:

                        
 #include <stdio.h>

 // Variable declaration:
 extern int a, b;
 extern int c;
 extern float f;

 int main ()
 {

     /* variable definition: */
     int a, b;
     int c;
     float f;

     /* actual initialization */
     a = 10;
     b = 20;

     c = a + b;
     printf("value of c : %d \n", c);

     f = 70.0/3.0;
     printf("value of f : %f \n", f);

     return 0;
 }
Conditional Statements

Decision making structures require that the programmer specifies one or more conditions to be evaluated or tested by the program, along with a statement or statements to be executed if the condition is determined to be true, and optionally, other statements to be executed if the condition is determined to be false.

The control flow statements are of following type:

  1. if statement
  2. if...else statement
  3. nested if statements
  4. switch statement
  5. The ? : Operator

The syntax of the if statement in C programming is:

                        
 if (test expression)
 {
 // code
 }

An example is as follows:

                        
 // Check whether an integer is odd or even

 #include <stdio.h>
 int main() {
     int number;
     printf("Enter an integer: ");
     scanf("%d", &number);

     // True if the remainder is 0
     if  (number%2 == 0) {
         printf("%d is an even integer.",number);
     }
     else {
         printf("%d is an odd integer.",number);
     }

     return 0;
 }

Output is:

                        
 Enter an integer: 7
 7 is an odd integer.
Loops

You may encounter situations, when a block of code needs to be executed several number of times. In general, statements are executed sequentially: The first statement in a function is executed first, followed by the second, and so on. Programming languages provide various control structures that allow for more complicated execution paths. A loop statement allows us to execute a statement or group of statements multiple times.

Types of loops present in C language:

  1. while loop
  2. for loop
  3. do...while loop

while loop

The syntax is as follows:

                        
 while(condition) {
 statement(s);
 }

Example: 

                        
 #include <stdio.h>

 int main () {

     /* local variable definition */
     int a = 10;

     /* while loop execution */
     while( a < 20 ) {
         printf("value of a: %d\n", a);
         a++;
     }

     return 0;
 }

for loop

The syntax is as follows:

                        
 for ( init; condition; increment ) {
 statement(s);
 }

Example: 

                        
 #include <stdio.h>

 int main () {

     int a;

     /* for loop execution */
     for( a = 10; a < 20; a = a + 1 ){
         printf("value of a: %d\n", a);
     }

     return 0;
 }

do..while loop

The syntax is as follows:

                        
 do {
 statement(s);
 } while( condition );

Example: 

                        
 #include 

 int main () {

     /* local variable definition */
     int a = 10;

     /* do loop execution */
     do {
         printf("value of a: %d\n", a);
         a = a + 1;
     }while( a < 20 );

     return 0;
 }
Functions

A function is a block of code that performs a specific task. Dividing a complex problem into smaller chunks makes our program easy to understand and reuse.

Types of function

  1. Standard library functions
  2. User-defined functions

User-defined function

You can also create functions as per your need. Such functions created by the user are known as user-defined functions.

Syntax of User Defined Functions:

                        
 #include <stdio.h>
 void functionName()
 {
     ... .. ...
     ... .. ...
 }

 int main()
 {
     ... .. ...
     ... .. ...

     functionName();

     ... .. ...
     ... .. ...
 }

An example is as follows:

                        
 /* function returning the max between two numbers */
 int max(int num1, int num2) {

     /* local variable declaration */
     int result;

     if (num1 > num2)
         result = num1;
     else
         result = num2;

     return result;
 }
